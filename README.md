## Project Title

##  Type

 ## Current Status

 ### Technologies Used

- React
- Node.js
- Express
- MongoDB
- Vercel


### Features
1. **User Authentication**: Secure email/password registration and login with Firebase.
2. **Car Browsing**: Browse available cars with detailed specifications, images, and pricing.
3. **Car Search & Filtering**: Search by brand, model, price range, mileage, and year.
4. **Car Listing**: Users can create listings to sell their cars with images and full details.
5. **Admin Dashboard**: Manage all car listings, users, and system settings.
6. **Responsive Design**: Built with Tailwind CSS for seamless experience on desktop, tablet, and mobile devices.
7. **Cloud Storage**: Automatic image uploads to Cloudinary with secure storage and delivery.
